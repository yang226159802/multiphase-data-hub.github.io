# Multiphase Data Hub v0.1 Release Notes

This v0.1 release establishes the first public structure of Multiphase Data Hub.

## Included

- Static GitHub Pages website.
- Seed dataset record for droplet breakup in homogeneous isotropic turbulence.
- ModelScope as the first primary data host.
- Dataset catalogue page.
- Data standard page.
- Contribution workflow.
- Hosting guide for ModelScope, Kaggle, and Zenodo.
- Governance and review page.
- Dataset submission issue template.
- Pull request template.

## Not Included Yet

- Backend upload system.
- User login.
- Automated metadata validation.
- Automated link checking.
- Complete benchmark leaderboard.

## Intended Feedback

Early reviewers are invited to comment on:

- clarity of the data standard;
- whether the contribution workflow is practical;
- missing metadata fields for multiphase-flow simulations;
- preferred hosting platforms;
- additional seed datasets.
