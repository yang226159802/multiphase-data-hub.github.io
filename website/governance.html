Governance

Lightweight review for reusable data

The catalogue does not certify scientific perfection. It checks that datasets are understandable, citable, licensed, and practically usable.

Metadata Review

Variables, units, grid, file paths, phase convention, and snapshot information must be clear.

Access Review

Download links must work, and hosted files must match the metadata record.

License Review

The dataset must state how others may reuse, cite, and redistribute the files.

Reproducibility Review

Solver-specific binary files must include enough information for external users to read them.
