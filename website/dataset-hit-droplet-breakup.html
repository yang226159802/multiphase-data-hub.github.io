HIT Droplet Breakup: Gvol, Gcurv, and Velocity

A raw EnSight Gold subset from a three-dimensional droplet breakup case in homogeneous isotropic turbulence. The first public entry is hosted on ModelScope, with Kaggle planned as an international mirror.

Variables

Gvol volume fraction, Gcurv interface curvature, and V velocity vector.

Resolution

Structured 3D grid with 256 x 256 x 256 cells and EnSight Gold binary geometry.

Snapshots

67 selected files from the original 401 snapshots: 000001, 000007, 000013, ...

Access

Hosted on ModelScope. Kaggle mirror planned for international access.

Download

Access the released case on ModelScope.

The data files are hosted externally. This website provides the dataset description, metadata, and stable entry point.

Open on ModelScope Metadata JSON

Raw Format

EnSight files are preserved as the released data format.

Case File

arts_subset.case defines geometry, variables, file patterns, and selected time values.

Grid

geometry stores the EnSight Gold binary structured geometry shared by all selected snapshots.

Data Folders

Gvol/, Gcurv/, and V/ contain the selected raw variable files.

Metadata

info.json and multiphase_info.json document variables, grid, snapshot selection, and file paths.

Interface Geometry

Curvature and phase-indicator fields are included with velocity.

The subset is intended as a first public example of how raw multiphase-flow simulation outputs can be packaged without forcing every contributor into a single AI-ready tensor format.
