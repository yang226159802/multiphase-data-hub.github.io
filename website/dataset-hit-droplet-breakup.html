Droplet breakup in HIT

Direct numerical simulation data for a liquid droplet breaking up in forced homogeneous isotropic turbulence.

Description

This dataset contains direct numerical simulation data of a deforming and breaking droplet in homogeneous isotropic turbulence, obtained using a conservative level set interface-capturing scheme. The simulation is performed in a periodic cubic domain of size (2pi)^3, with turbulence resolved on a uniform Cartesian grid of 256^3 points. The Taylor-scale Reynolds number is 100, the Weber number is 15, and the initial droplet diameter is D = pi. Both the density ratio and viscosity ratio between the two phases are equal to unity. The dataset provides time-resolved velocity, pressure, and interface fields for this representative droplet breakup case.
