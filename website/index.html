Why this exists

Multiphase-flow AI needs shared data, not only shared models.

High-fidelity two-phase and multiphase-flow simulations are expensive, experimental data are hard to standardize, and many AI papers rely on private case-specific datasets. This site turns scattered datasets into citable, searchable, and reusable research assets.

Submission model

Keep the website light. Put large data on archival storage.

1

Host

Upload data to ModelScope for China-accessible hosting, Kaggle for international access, or Zenodo for DOI-backed archives.

2

Describe

Fill the metadata template with physics, numerics, variables, units, splits, license, and citation.

3

Review

Submit a pull request. Maintainers check metadata completeness, link integrity, and license clarity.

See the hosting guide for the short ModelScope, Kaggle, and Zenodo workflows used by this project.
