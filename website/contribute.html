Datasets Standard Contribute

Contribute

Submit datasets without handing the data to the website itself.

Contributors host large data on stable storage and submit metadata to this catalogue. The review focuses on clarity, license, provenance, and reproducibility.

1

Prepare

Package data with a README, metadata JSON, checksums, and at least one loading example.

2

Archive

Upload to ModelScope for China-accessible hosting, Kaggle for international access, or Zenodo for archival DOI releases.

3

Submit

Open a GitHub pull request that adds one JSON file under the catalogue datasets folder.

Acceptance checklist

Minimum bar for catalogue inclusion

  • Dataset has an explicit data license.
  • Dataset has a stable URL or DOI.
  • Variables, units, array shapes, and coordinate order are documented.
  • Simulation or experimental provenance is described.
  • AI datasets include split policy and leakage-prevention notes.
  • At least one external user can load a sample with the provided documentation.

Template

Metadata starts from a JSON record.

Use the repository template at dataset_info.template.json. The schema is available at dataset_info.schema.json.
