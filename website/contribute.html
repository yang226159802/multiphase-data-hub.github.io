Contribute

Submit a dataset record, not the data files themselves.

Contributors host large files on ModelScope, Kaggle, Zenodo, or institutional storage. This catalogue reviews and publishes metadata, access links, documentation, and citation information.

1

Package

Prepare data files, README.md, info.json, checksums, and a reader note or example script.

2

Host

Upload to ModelScope for China-accessible distribution, Kaggle for international access, or Zenodo for DOI-backed release.

3

Submit

Open a GitHub issue or pull request with the metadata record, preview figure, download link, license, and citation.

Acceptance Checklist

Minimum bar for catalogue inclusion

  • Data are hosted outside this website repository.
  • Download or landing-page URL is public or clearly access-controlled.
  • License and citation are explicit.
  • Variables, units, shapes, and field locations are documented.
  • Grid, coordinates, boundary conditions, and phase convention are described.
  • Solver-specific binary files include header, dtype, payload shape, and reading instructions.
  • At least one representative image or preview figure is provided.
