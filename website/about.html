About

An open catalogue for multiphase-flow datasets

Multiphase Data Hub is a community-oriented effort to make multiphase-flow simulation and experimental datasets easier to find, cite, download, and reuse.

Motivation

Shared data is becoming infrastructure for AI-assisted CFD.

High-fidelity multiphase simulations and experiments are expensive, and many datasets remain private or difficult to interpret. This site provides a lightweight metadata catalogue and contribution workflow inspired by BlastNet, adapted to multiphase-flow variables, grid descriptions, and solver-native formats.

v0.1 Scope

The first release focuses on curated dataset records, metadata standards, and hosting guidance. It does not host large data files directly and does not provide a login/upload backend.
